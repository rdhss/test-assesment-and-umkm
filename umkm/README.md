## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rdhss/test-assesment-and-umkm.git
   ```
2. Go to UMKM 
   ```sh
   cd umkm
   ```
3. Install npm
   ```sh
   npm install
   ```
4. Run Application
   ```sh
   npm start
   ```
